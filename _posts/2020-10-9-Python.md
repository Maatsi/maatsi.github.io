# Enumerate
- (count, element) in enumerate (list)
- **Enumerate** iterates through the list and also counts the current iteration.

```
# list object
list = ['boo','bam','baa']   

for (count,element) in enumerate (list):
  print (count, element)
```

Output:
> 0, 'boo'  
> 1, 'bam'  
> 2, 'baa'
